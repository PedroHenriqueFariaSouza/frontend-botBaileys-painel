# React no Projeto

## Papel no projeto

React e a base da interface. Ele organiza o painel em componentes reutilizaveis e facilita a separacao entre telas como bots, pareamento, usuarios e comandos.

## Onde aparece no codigo

- src/App.tsx para roteamento simples por estado
- src/pages para telas principais
- src/components para partes reutilizaveis (exemplo: sidebar)

## Importancia pratica

- Permite evoluir funcionalidades sem acoplamento excessivo
- Facilita manutencao de estado visual por pagina
- Melhora produtividade no desenvolvimento de UI administrativa

## Decisoes relevantes

- Fluxo de navegacao interno sem dependencias extras de roteamento
- Composicao por paginas para manter clareza da arquitetura frontend
