# Material UI no Projeto

## Papel no projeto

Material UI fornece a base visual do painel, com componentes acessiveis e consistentes para formularios, tabelas, dialogs, alerts e navegacao.

## Onde aparece no codigo

- src/theme.ts com tema global
- src/pages e src/components para layout e componentes interativos

## Importancia pratica

- Acelera construcao de interfaces administrativas
- Mantem consistencia visual entre paginas
- Reduz codigo manual de estilo para componentes comuns

## Decisoes relevantes

- Uso de componentes padrao como Table, Dialog, Snackbar, Alert e Drawer
- Tema centralizado para facilitar evolucao visual futura
